# Run & screenshot the Clubul Financiar site

The site is a static GitHub Pages site served from `docs/` (CNAME `clubulfinanciar.ro`).
There is no build server ŌĆö pages are plain HTML + `assets/*.css` + `assets/*.js`.
Some pages are generated by Python builders (`_build_site.py`, `_build_glosar.py`,
`_build_manual.py`, `_build_masterclass.py`, `_build_ai_knowledge.py`); editing those
pages' look means editing the shared CSS/shell, not the generated HTML.

## One command (preferred)

```bash
cd ~/clubul-financiar && ./_screenshot.sh <label> [page1.html page2.html ...]
```

- `<label>` groups the batch into `ŌĆ”/scratchpad/shots/<label>/`.
- With no pages it shoots a representative set (index, premium, educatie,
  calculatoare, instrumente, masterclass, stiri, glosar, incepe-aici, login).
- Produces `<page>__1440.png` (desktop) and `<page>__390.png` (mobile) per page.
- Starts a `python3 -m http.server` on port 8765, screenshots, then kills it.
- Override output dir with `CF_SHOTDIR=ŌĆ”`, port with `CF_PORT=ŌĆ”`.

After it prints `==> <dir>`, **Read the PNGs** ŌĆö a blank/half frame means the page
failed to render (often a JS/Three.js error), not a pass.

## Manual fallback

```bash
cd ~/clubul-financiar/docs
python3 -m http.server 8765 >/tmp/cf_server.log 2>&1 &      # serve
curl -s -o /dev/null -w "%{http_code}\n" http://localhost:8765/index.html  # smoke
CHROME="/Applications/Google Chrome.app/Contents/MacOS/Google Chrome"
"$CHROME" --headless=new --use-angle=swiftshader --disable-gpu --hide-scrollbars \
  --window-size=1440,2600 --screenshot=/tmp/cf.png http://localhost:8765/index.html
lsof -ti:8765 | xargs kill -9                                # stop
```

## Notes / gotchas

- Use a real HTTP server, not `file://` ŌĆö the homepage uses a Three.js importmap,
  webfonts, and `fetch()` for live data (news/fx/macro); `file://` breaks them.
- `--use-angle=swiftshader` is required for the WebGL 3D hero to render headless.
- The 3D hero canvas (`#hero3d` via `assets/three3d.js`) needs a moment; the
  screenshot at default timing usually captures it. If it's blank, the WebGL flag
  is missing or Three.js failed to load.
- Dark theme: the site reads `localStorage["cf-theme"]`; headless starts in the
  default (light/cream) theme.
- Mobile nav collapses below 980px; the burger (`#burger`) drives `.nav-links.open`.
