Masa de lucru
**Tensiunile Iran‑SUA și așteptările Fed pot rupe pierderea în piețele financiare. Și dacă țițeiul urcă peste 78 USD/baril, există o strategie simplă de protecție pentru cei care lucrează cu PFA sau cu o SRL.**
Chemarea zilei
**Decizie:** *Dezvolta 30 % din profitul curent al PFA într-un depozit bancar cu dobândă variabilă și restul în titluri de stat cu dobândă fixă.*
**Argument matematic simplu:**
- Dobânda medie depozitului = **3,5 %** anual (bănci românești, 2024).
- Titluri de stat 10‑ani (RO) ofere **5,8 %** anual, dar cu risc de volatilitate dacă ratele de dobânză se ridică.
- Dacă inflația previsible este **3,2 %**, diferența netă este:
- Deposit: **3,5 % – 3,2 % = 0,3 %** real.
- Titlu de stat: **5,8 % – 3,2 % = 2,6 %** real.
- Pentru un profit de **10.000 EUR** în PFA, alocarea 30 % în depozito (3.000 EUR) generează **9 EUR** pe an real, în timp ce 7.000 EUR în titlu de stat generează **182 EUR** pe an real.
- Dacă ratele de dobânză se crestesc cu **0,5 %** (scenario inflație + rate), diferența devine:
- Deposit: **4,0 % – 3,2 % = 0,8 %** real → **24 EUR** pe an.
- Titlu de stat: **6,3 % – 3,2 % = 3,1 %** real → **217 EUR** pe an.
- **Concluzie:** cu un singur pași simplu, poți fixa un buffer de siguranță (depozit) și să beneficiezi de rata mai mare a titlurilor de stat, fără a expozi întreaga profit la volatilitate.
Acționabil PFA/SRL
**Termeni fiscali și praguri pentru RO**
- **Dobânzile de dobânză pe depozite** sunt taxate la **10 %** pe diferența dobânzilor (nu pe capital).
- **Titluri de stat** (Obligații de Stat România) generă **10 %** de taxă pe venituri din dobânză, cu excepție pentru persoanele fizice care reinvestesc în SRL‑uri de economii.
- **Cheltuieli deducibile** pentru PFA:
- 100 % din cheltuielile de administrare (software, birou, internet) până la **2.500 EUR** pe an.
- 50 % din cheltuielile de reprezentare (conferințe, seminari) cu limită de **1.500 EUR** pe an.
- **Cotație de depreciere** pentru echipament SRL: 20 % pe bază anuară pentru calculatoare și aparatătechnică, 10 % pentru vehicule.
- **Rata de parteneriat** pentru SRL: dacă venitul anual este sub **500.000 EUR**, se aplică **16 %** de impôt pe venit (cu 2 % de contributions sociale).
- **Strategie practică:**
1. Deschide un cont de depozit cu dobândă variabilă și depune **30 %** din profitul curent.
2. Alocă restul în obligații de stat cu dobândă fixă (ex. 10‑ani, cu dobânda 5,8 %).
